1) install python on your computer
2) install pip on your device and make sure to use last version
3) run your powershell and use this command:
winget install --id Git.Git -e --source winget
this will install git on your device
4) cmd and type this comands and run them one by one or insert all of them once:
pip install git+https://github.com/Joeclinton1/google-images-download.git
pip install kivy
pip install pygame
pip install bing-image-downloader
5) copy the [source code](https://github.com/FlatAd560/ImageDownloaderBuddy/blob/main/sources/IDBwindows.py) and
paste it in your python app (if you using default python programm, you have to create a new file first)
6) run this code, if there's no problem ahead, please check the user guide
(https://github.com/FlatAd560/ImageDownloaderBuddy/blob/main/UserGuideWindows.txt),
if you have found a problem, please check the commands above and make sure all of them installed successfully
or leave a issue in github.
